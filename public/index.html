Request Header Parser

This request header parser will take your request to the server and return to you your useragent, language, and ip address.

Enter this URL into your browser bar to see your machine's information:

https://matty22requestheaderparser.herokuapp.com/whoami

Example output:

{"useragent":"Mozilla/5.0 (Macintosh; Intel Mac OS X 10_12_0) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/54.0.2840.98 Safari/537.36","language":"enUS,en;q=0.8", "ip":"71.35.101.10"}
